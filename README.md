# LLM for Rap Generation
The paper "Large Language Model for Rap Generation" and code to reproduce its results. Check our final model DistilGPT2-rap on [Hugging Face](https://huggingface.co/dzionek/distilgpt2-rap).

Authors:
- [Bartosz Dzionek](https://github.com/dzionek)
- [Jamie Welsh](https://github.com/jamiewelsh25)
- [Emmy Zhou](https://github.com/3mmyzhou)

Work done as part of the *Machine Learning Practical* course at the University of Edinburgh.
